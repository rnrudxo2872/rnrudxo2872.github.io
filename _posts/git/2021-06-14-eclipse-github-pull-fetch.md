## pull? fetch?

---

간단히 말하자면, **fetch**는 로컬 저장소에 원격 저장소 **커밋들을 가져오는 것**이고, **pull**은 원격 저장소의 커밋들을 가져와서 변경되어 있다면, 병합하여 업데이트, HEAD를 같은 위치로 만든다. 민감한 작업에는 fetch로 미리 변경사항을 확인하고 작업을 진행하는게 좋다.
